# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-25

### Added
- Initial release of Avo Counter
- Per-page visit tracking
- Last updated date display
- Admin dashboard showing page statistics
- Shortcode [avo_counter] for displaying counter
- Clean card-based design with blue accent bar
- Mobile responsive layout
- Hover effects on counter card
- Settings page for customizing counter start date
- Support for pages and posts

### Features
- Lightweight, no external dependencies
- Uses WordPress post meta for storage
- Automatic date tracking on page updates
- Admin panel with statistics table
- Customizable start date

### Technical
- WordPress 5.0+ compatible
- PHP 7.2+ required
- GPL2 licensed
